# Projet PPSPS

Outil d’automatisation de documents Word standardisés.

## Fonctionnalités actuelles
- Chargement de fichiers `.docx`
- Détection et édition de balises `{{...}}`
- Exportation du document modifié avec ou sans balises

## Structure du projet
- `index.html` → structure du site
- `main.js` → logique principale
- `libs/` → bibliothèques externes
- `assets/` → images, logos, icônes

---

Projet conçu pour fonctionner 100 % sur navigateur, sans serveur.
